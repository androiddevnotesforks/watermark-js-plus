# Tools

Use these browser-based tools to complete common watermark tasks without writing integration code first.

## [Watermark Configurator](/tools/watermark-configurator)

Adjust watermark options interactively, preview the result on the page, and copy the generated initialization code.

## [Blind Watermark Decoder](/tools/blind-watermark-decoder)

Upload or paste an image, tune the decode options, and reveal its blind watermark.
